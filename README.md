# ✨ Raj Life - Life Manager App

A beautiful, colorful Progressive Web App (PWA) to manage your complete life - finances, daily tasks, and life goals!

## 🚀 Features

### 📊 Dashboard
- Complete overview of your financial health
- Task completion progress with visual charts
- Goals achievement tracking
- Colorful bar charts and progress indicators
- Today's tasks at a glance

### 💰 Finance Manager
- **Income Tracking** - Track salary and regular income
- **Expense Tracking** - Monitor where your money goes
- **Loan Management** - Track who you owe and who owes you
- **Credit Card Bills** - Keep credit card expenses in check
- **Other Income** - Freelance, gifts, investments etc.
- Add & Delete entries with full details

### ✅ Task Manager
- **Daily Tasks** - Things to do today
- **Weekly Tasks** - Plan your week
- **Priority System** - High, Medium, Low priority
- **Filters** - View All, Daily, Weekly, Done, Pending
- Mark as Done / Undo
- Beautiful progress bar

### 🎯 Life Goals
- Set and track life goals
- **Categories**: Personal, Career, Health, Finance, Education, Travel, Relationship
- Progress tracking (10%, 25% increments)
- Deadline setting
- Mark as achieved / reopen
- Motivational quotes

## 📱 Works Everywhere!
- **Phone** - Install as app (PWA)
- **Laptop/Desktop** - Full responsive design
- **Data Sync** - Uses localStorage (same browser)
- **Offline Support** - Works without internet

## 🛠️ Tech Stack
- React 18 + Vite
- Tailwind CSS (glassmorphism design)
- PWA (installable on phone)
- LocalStorage for data persistence

## 🎨 Design
- Dark theme with glassmorphism
- Gradient colors (Indigo, Purple, Pink)
- Smooth animations
- Mobile-first responsive design
- Beautiful emoji icons

## 📦 Setup

```bash
cd raj-life
npm install
npm run dev
```

## 🌐 Deploy
Build for production:
```bash
npm run build
```
Deploy the `dist` folder to any hosting (Vercel, Netlify, GitHub Pages).
